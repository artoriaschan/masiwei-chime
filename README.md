# masiwei-chime
> 马思唯整点报时

## features
- [ ] 自定义模板
- [ ] 报时语音播报
- [x] 自定义报时名称
- [x] 自定义时间格式
- [x] 可复制提示语句
- [x] 可追加首部和尾部增加文字

## Configuration
### msw.name
报时人名称
### msw.timeFormat
报时时间格式

参照dayjs的[Format](https://dayjs.gitee.io/docs/zh-CN/display/format)
### msw.extra
报时模板追加首部和尾部增加文字
### msw.customTemplateArray
增加自定义模板(未实现)
### msw.autoSpeech
是否自动语音播报(未实现)
